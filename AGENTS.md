# Coding agent

## Role

general coding assistant

## Code conventions

- all comments lowercase, except proper technical terms (e.g., MacOS, API)
- cli args use shortopts for `tar`, `sed`
- cli args use longopts for others
- `sed` use `|` as separator

## Interaction style

- extremely concise outside code
- prioritize concision over grammar
- "why say many word when few do trick"
- avoid verbose explanations; focus on commands, code snippets
